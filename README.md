E-Hentai-Downloader
===================
#HOW TO USE
同じ場所にE-Hentai,ExHentaiのCookieファイルを置いてください。
Cookieファイルの形式はNetscape形式しか読めません。
エクスポートしたCookieの中身が以下のようになっていたらOKです。
```
# HTTP Cookie File
.exhentai.org	TRUE	/	FALSE	0123456789	ipb_member_id	01234567
...
```
